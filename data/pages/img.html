Словарная статья: тег <img> v0.0

Словарная статья в онлайн‑словаре по HTML и CSS

Определение

<img> — элемент для вставки изображений в веб‑страницу. Отображает графическое содержимое по ссылке на файл. Является пустым (не имеет закрывающего тега) и относится к встроенным (inline) элементам.

Синтаксис

<img src="путь/к/изображению.jpg" alt="описание изображения" width="100" height="150">

Пример использования

Изображение товара в карточке:

Смартфон XYZ Pro

Современный смартфон с тройной камерой и мощным процессором.

Код примера:

<div class="product-card">
  <h3>Смартфон XYZ Pro</h3>
  <img src="images/smartphone-xyz-pro.jpg"
       alt="Смартфон XYZ Pro, вид спереди"
       width="300"
       height="400">
  <p>Современный смартфон с тройной камерой...</p>
</div>

Основные атрибуты

Атрибут Назначение Пример
src Обязательный атрибут. Указывает путь к изображению (URL или локальный путь) src="images/photo.jpg"
alt Текстовое описание изображения. Критично для доступности и SEO alt="Логотип компании"
width Ширина изображения в пикселях или процентах width="500"
height Высота изображения в пикселях или процентах height="300"
loading Управляет загрузкой: lazy (отложенная загрузка) или eager (обычная) loading="lazy"
srcset Указывает несколько версий изображения для разных разрешений экрана srcset="small.jpg 480w, large.jpg 1080w"

Связанные термины

Атрибут alt Адаптивные изображения Форматы изображений (JPEG, PNG, WebP) <picture> Доступность (a11y)

Примечания

  • Атрибут alt обязателен: без него страница не пройдёт проверку на доступность.
  • Для фоновых изображений используйте CSS (background-image), а не <img>.
  • Используйте loading="lazy" для изображений ниже первого экрана — это ускоряет загрузку страницы.
  • Всегда указывайте width и height, чтобы избежать «прыжков» контента при загрузке.
  • Современные форматы (WebP, AVIF) позволяют уменьшить размер файла без потери качества.
  • Тег <img> не поддерживает содержимое между открывающим и закрывающим тегом — это самозакрывающийся элемент.
