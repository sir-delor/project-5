Определение

text-align — CSS‑свойство, определяющее горизонтальное выравнивание текста внутри элемента. Применяется к блочным контейнерам (<div>, <p>, заголовкам и т. д.) и влияет на все строчные элементы внутри них.

Синтаксис

text-align: left | right | center | justify | start | end;

Пример использования

Разметка с разными вариантами выравнивания текста:

По левому краю
Текст выровнен по левому краю — стандартное значение.

По центру
Текст центрирован относительно контейнера.

По правому краю
Текст выровнен по правому краю контейнера.

Код примера:

/* Стили для демонстрации */
.examples {
    display: flex;
    gap: 20px;
    margin: 20px 0;
}

.example-box {
    width: 30%;
    border: 1px solid #ccc;
    padding: 10px;
}

.left-align {
    text-align: left;
}

.center-align {
    text-align: center;
}

.right-align {
    text-align: right;
}

/* HTML разметка */
<div class="examples">
    <div class="example-box left-align">
        <p class="example-text">Текст выровнен по левому краю</p>
    </div>
    
    <div class="example-box center-align">
        <p class="example-text">Текст центрирован</p>
    </div>
    
    <div class="example-box right-align">
        <p class="example-text">Текст выровнен по правому краю</p>
    </div>
</div>

Основные значения

Значение Описание Пример
left Выравнивание по левому краю (значение по умолчанию для LTR‑языков) text-align: left;
right Выравнивание по правому краю text-align: right;
center Центрирование текста относительно контейнера text-align: center;
justify Растягивает строки так, чтобы они занимали всю ширину контейнера (как в газетах) text-align: justify;
start Выравнивание по началу текста (зависит от направления письма: left для LTR, right для RTL) text-align: start;
end Выравнивание по концу текста (противоположно start) text-align: end;

Связанные термины

CSS Box Model vertical-align Flexbox (justify-content) Grid Layout Направление текста (direction)

Примечания

  • Свойство text-align применяется только к блочным элементам. Для выравнивания строчных элементов (<span>) нужно обернуть их в блок.
  • Значение justify может создавать большие пробелы между словами — используйте осторожно.
  • В RTL‑языках (direction: rtl;) значения start/end меняются местами относительно left/right.
  • Для вертикального выравнивания используйте vertical-align (для таблиц и inline‑элементов) или Flexbox/Grid.
  • При использовании text-align: center; для центрирования блока лучше применять margin: auto; или Flexbox.
  • Браузеры поддерживают значения start и end с учётом спецификации CSS Text Level 3.
