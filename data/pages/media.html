Словарная статья в онлайн‑словаре по HTML и CSS

Определение

@media — CSS‑директива для реализации адаптивного дизайна. Позволяет применять стили в зависимости от характеристик устройства или области просмотра (ширина, высота, ориентация, разрешение и т. д.).

Синтаксис

@media условие {
  /* CSS‑правила */
}

Пример использования

Адаптация макета для мобильных устройств:

Мобильная версия сайта

На экранах шириной до 768 px меню сворачивается в «бургер», а отступы уменьшаются.

Код примера:

@media (max-width: 768px) {
  .nav-menu {
    display: none;
  }
  .burger-menu {
    display: block;
  }
  body {
    padding: 10px;
  }
}

Основные условия (медиа‑функции)

Условие Назначение Пример
max-width Применяется, если ширина области просмотра ≤ указанного значения (max-width: 600px)
min-width Применяется, если ширина области просмотра ≥ указанного значения (min-width: 1200px)
orientation Ориентация экрана: portrait или landscape (orientation: portrait)
resolution Разрешение экрана (dpi, dppx) (resolution: 300dpi)
prefers-color-scheme Предпочитаемая цветовая схема пользователя (dark/light) (prefers-color-scheme: dark)

Логические операторы

  • and — объединяет условия (все должны быть истинными).
  • not — инвертирует условие.
  • , (запятая) — работает как «ИЛИ»: достаточно выполнения одного условия.
/* Ширина от 600 до 1200 px И портретная ориентация */
@media (min-width: 600px) and (max-width: 1200px) and (orientation: portrait) { ... }

/* Тёмная тема ИЛИ экран с высоким разрешением */
@media (prefers-color-scheme: dark), (resolution: 2dppx) { ... }

Примечания

  • Располагайте медиа‑запросы после основных стилей, чтобы избежать конфликтов переопределения.
  • Используйте em вместо px для относительных брейкпоинтов — это улучшает масштабирование.
  • Тестируйте адаптивность в инструментах разработчика браузера (например, в Chrome DevTools).
  • Для печати используйте @media print, чтобы скрыть навигацию, рекламу и т. п.
  • Избегайте избыточных медиа‑запросов: оптимизируйте код, объединяя схожие условия.
