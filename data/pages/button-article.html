Определение

<button> — интерактивный элемент интерфейса, создающий кнопку. Используется для выполнения действий при клике: отправки форм, запуска скриптов, навигации и т. д. В отличие от ссылок (<a>) предназначен для действий внутри страницы.

Синтаксис

<button type="button" class="имя-класса" id="уникальный-id">
  Текст кнопки
</button>

Пример использования

Набор кнопок для интерфейса корзины покупок:

Код примера:

<div class="cart-actions">
  <button type="button">Добавить в корзину</button>
  <button type="button" disabled>Товар в корзине</button>
  <button type="submit">Оформить заказ</button>
</div>

Основные атрибуты

Атрибут Назначение Пример
type Тип кнопки: button (обычная), submit (отправка формы), reset (сброс формы) type="submit"
disabled Блокирует кнопку, делая её неактивной disabled
name Имя кнопки для передачи данных в форме name="add-to-cart"
value Значение кнопки, отправляемое на сервер value="Добавить"

Связанные термины

Форма (<form>) События JavaScript Атрибут disabled Семантические элементы

Примечания

  • По умолчанию имеет встроенное оформление браузера (рамка, фон, эффекты при наведении).
  • Для кастомного дизайна полностью переопределяйте стили через CSS.
  • Используйте type="button" для кнопок без стандартной формы — это предотвращает случайную отправку.
  • Обязательно добавляйте атрибут aria-label для доступности, если текст кнопки не очевиден.
  • В формах кнопки type="submit" автоматически отправляют данные, если не перехвачены JavaScript.
