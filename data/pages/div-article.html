Определение

<div> — блочный элемент‑контейнер общего назначения, используемый для группировки других элементов и применения к ним стилей CSS. Не несёт семантической нагрузки — предназначен исключительно для структурирования разметки.

Синтаксис

<div class="имя-класса" id="уникальный-id">
  <!-- Содержимое -->
</div>

Пример использования

Карточка товара, созданная с помощью <div>:

Смартфон XYZ Pro

Современный смартфон с тройной камерой и мощным процессором.

Цена: 49 999 руб.

Код примера:

<div class="example-card">
  <h3>Смартфон XYZ Pro</h3>
  <p>Современный смартфон с тройной камерой...</p>
  <div class="price">Цена: 49 999 руб.</div>
  <button>В корзину</button>
</div>

Основные атрибуты

Атрибут Назначение Пример
class Присваивает элемент классу CSS для стилизации class="card"
id Задаёт уникальный идентификатор элемента id="main-content"
style Встроенные стили (не рекомендуется для больших проектов) style="color: red;"

Связанные термины

CSS‑классы Селектор по ID Блочная модель Flexbox

Примечания

  • <div> по умолчанию занимает всю доступную ширину родителя (display: block).
  • В современных проектах рекомендуется использовать семантические теги вместо <div>, если это соответствует смыслу контента.
  • Часто используется как контейнер для Flexbox или Grid‑макетов.
