Определение

<footer> — HTML‑тег, определяющий нижний колонтитул (подвал) секции или всего документа. Обычно содержит информацию об авторских правах, ссылки на разделы сайта, контактную информацию и т. д.

Синтаксис

<footer>
  <!-- Содержимое подвала -->
</footer>

Пример использования

Подвал всего сайта:

© 6 Мой сайт. Все права защищены.

О нас | Контакты | Политика конфиденциальности
VK Telegram

Код примера:

<footer>
  <p>© 2026 Мой сайт. Все права защищены.</p>
  <nav>
    <a href="/about">О нас</a> |
    <a href="/contact">Контакты</a> |
    <a href="/privacy">Политика конфиденциальности</a>
  </nav>
  <div class="social">
    <a href="#">VK</a>
    <a href="#">Telegram</a>
  </div>
</footer>

Подвал статьи:

Заголовок статьи

Текст статьи...

Автор: Иван Иванов

Опубликовано: 17 апреля 2026

Код примера:

<article>
  <h2>Заголовок статьи</h2>
  <p>Текст статьи...</p>
  <footer>
    <p><small>Автор: Иван Иванов</small></p>
    <p><small>Опубликовано: 17 апреля 2026</small></p>
  </footer>
</article>

Основные атрибуты

Атрибут Назначение Пример
class Присваивает элементу класс CSS для стилизации class="site-footer"
id Задаёт уникальный идентификатор элемента id="main-footer"
style Встроенные стили (не рекомендуется для больших проектов) style="background-color: #f0f0f0;"

Связанные термины

<header> <nav> <main> <section> <article>

Примечания

  • <footer> — семантический тег HTML5, логически обозначает «завершающую часть».
  • В документе может быть несколько тегов <footer>.
  • Не обязательно должен находиться в самом низу разметки — его положение определяется логикой структуры документа.
  • По умолчанию браузеры не применяют к <footer> особых стилей.
  • Программы экранного чтения распознают <footer> и могут предоставлять пользователям быстрый доступ к нему.
